# Jawafdehi Agents

Flyte-based agent orchestration service for Jawafdehi workflows.

## Development

Install dependencies with Poetry:

```bash
poetry install
```

Run tests:

```bash
poetry run pytest
```

Format and lint:

```bash
./scripts/format.sh
./scripts/format.sh --check
```

Run the CLI:

```bash
poetry run jawaf run 081-CR-0046
```
